# graphql-apollo-server

## Getting Started

1. install

```zsh
  npx prisma migrate dev --name init
  npx prisma db seed --preview-feature
```

2. start

```zsh
  npm run dev
  # http://localhost:4000

  # prisma studio
  npx prisma studio
```
